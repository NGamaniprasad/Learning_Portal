1. Introduction to Java

Java is a high-level, object-oriented programming language used for building robust applications.

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, Java!");
  }
}

2. Variables and Data Types

Java supports various data types like int, float, String, boolean, etc.

int age = 25;
String name = "Alice";
float price = 99.99f;
boolean isJavaFun = true;

3. Control Statements (if/else)

Used to control the flow of execution based on conditions.

int marks = 85;
if (marks >= 90) {
  System.out.println("Grade A");
} else {
  System.out.println("Grade B");
}

4. Loops (for/while)

Used for executing a block of code repeatedly.

for (int i = 0; i < 5; i++) {
  System.out.println("Loop " + i);
}

int j = 0;
while (j < 3) {
  System.out.println("While loop " + j);
  j++;
}
