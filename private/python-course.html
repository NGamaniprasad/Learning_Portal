1. Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability.

Syntax:

print("Hello, Python!")

Sample Code:

# This is a comment
print("Welcome to Python programming!")

2. Variables and Data Types

Variables store data values. Python has various data types like int, float, string, list, etc.

Syntax:

x = 5
name = "Alice"
pi = 3.14

Sample Code:

age = 25
height = 5.9
name = "John"
print("Name:", name)

3. Control Flow (if/else)

Control structures like if, elif, else are used for decision-making.

Syntax:

if condition:
    # code
elif another_condition:
    # code
else:
    # code

Sample Code:

age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

4. Loops (for/while)

Loops are used to execute a block of code repeatedly.

Syntax:

for i in range(5):
    print(i)

while condition:
    print("Looping...")

Sample Code:

for i in range(3):
    print("Iteration", i)
